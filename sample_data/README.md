# 테스트용 샘플 데이터

이 디렉토리에는 고객 상담 프로그램 테스트를 위한 샘플 데이터가 포함되어 있습니다.

## 텍스트 파일

다음 텍스트 파일들이 포함되어 있습니다:

1. **sample_consultation_1.txt** - 배송 지연 및 환불 요청 사례
2. **sample_consultation_2.txt** - 제품 불만 및 A/S 요청 사례
3. **sample_consultation_3.txt** - 환불 및 교환 요청 사례
4. **sample_consultation_4.txt** - 기술 지원 요청 사례

## 사용 방법

1. 프론트엔드에서 파일 업로드 기능을 사용하여 이 텍스트 파일들을 업로드하세요.
2. 제목을 입력하고 파일을 선택한 후 "업로드 및 분석 시작" 버튼을 클릭하세요.
3. 분석이 완료되면 결과를 확인할 수 있습니다.

## 오디오/비디오 파일

현재 시스템은 오디오/비디오 파일도 지원합니다:
- **오디오**: MP3, WAV, M4A, OGG 형식
- **비디오**: MP4, AVI, MOV, WEBM 형식

오디오/비디오 파일의 경우 OpenAI Whisper API를 사용하여 자동으로 전사됩니다.

### 오디오/비디오 테스트 파일 생성 방법

실제 오디오/비디오 파일이 필요한 경우:
1. 자신의 상담 녹음 파일을 사용하거나
2. 온라인에서 샘플 오디오/비디오 파일을 다운로드하여 사용할 수 있습니다.

**참고**: 오디오/비디오 파일은 OpenAI Whisper API를 사용하여 전사되므로, OpenAI API 키가 필요합니다.
